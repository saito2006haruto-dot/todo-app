# ToDoアプリ

シンプルで直感的に使えるWebベースのToDo管理アプリです。  
タスクの追加・削除・検索・フィルタリングを通して、  
「日常のタスク管理を効率化すること」を目的として開発しました。

---

## リンク
https://saito2006haruto-dot.github.io/todo-app/

---

## 機能一覧

### 基本機能
- タスクの追加
- タスクの削除
- タスクの一覧表示

### 検索機能
- キーワードによる検索
- タスク内容の絞り込み

### 管理機能
- タスクの状態管理（未完了・完了など）

### データ保存
- LocalStorageによるデータ保存
